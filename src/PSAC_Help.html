Welcome to the PeopleSoft Admin Console.

This application gives you a central view of PeopleSoft application servers and the application servers, process schedulers and web servers

You'll be able to take actions against the services that are available: start up, shut down and interrogate the status

To initially import a list of servers in to the console you will need to create a flat file that has all the servers listed on seperate rows

Example

vmpshcmdev02
vmpsihdev01
vmpsfscmdev02

Save this as a .txt file in a place where you'll remember :)

From the menu select File > New, then choose the .txt file that contains the list of servers you wish to interrogate.

The application will then go have a poke around on those servers and see what PeopleSoft applications you have on there. If it finds anything it will display the details in the console

This might take some time as it will try to determine the status of the services that it finds: running, stopped, completely banjaxxed, etc.

Once you have a list that you would like to look at again (without having to wait for ages as it finds the services on them) you can use the File > Save As... option to save the list in XML format for future use

So you have a XML file that you've saved already eh? Well use the File > Open... menu item and get that loaded!

Once it's loaded the application will re-scan the services to see what their current state is

Now you have a list of servers and services they are providing you can use the actions available to look at what they are doing and actually tell them what to do!

Select one or more server/service from your list, select an action from the drop down list and hit the Go... button. If the action is relevant to the service then the application will try it's very best to do it!

You can also right click on the row to do more server specific actions like viewing (and killing off) the processes running, opening the Computer Management console and connecting to the server via RDP
